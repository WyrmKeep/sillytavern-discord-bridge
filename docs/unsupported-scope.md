# Unsupported In V1

- Group chats.
- Multi-character chats.
- Simultaneous editing of an active bridge chat in SillyTavern and Discord.
- Discord DMs.
- Attachments and image understanding.
- Streaming Discord replies.
- Public internet deployment.
- Alternate greeting selection.
- Live browser-only extension prompt state.
- Exact SillyTavern frontend token-budget parity.
