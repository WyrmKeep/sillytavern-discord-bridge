# Repository Agent Rules

- Follow `docs/plans/2026-04-17-discord-bridge-implementation.md`.
- Keep this repository private.
- Do not commit real Discord tokens, guild IDs, user IDs, SillyTavern settings, character cards, chat logs, or private proxy URLs.
- Use synthetic fixtures only.
- Preserve `package-lock.json`.
- Run `npm run typecheck`, `npm run lint`, `npm test -- --coverage`, and `npm run build` before completion claims.
- Implementation work belongs on feature branches, not directly on `main`.
