# SillyTavern Discord Bridge UI Extension

This branch is a root-level SillyTavern UI extension package for SillyTavern's built-in Git extension installer.

Install with:

```text
Git URL: https://github.com/WyrmKeep/sillytavern-discord-bridge
Branch or tag: st-ui-extension
```

This installs only the browser UI extension. The Discord bridge also requires the server plugin from the `main` branch to be installed under SillyTavern's `plugins/discord-bridge` directory with `enableServerPlugins: true`.
